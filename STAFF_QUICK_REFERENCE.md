## ğŸ†˜ Troubleshooting

### Problem: Can't find a request
**Solution:** 
1. Clear all filters
2. Check the "View" dropdown is set to "All Requests"
3. Try searching by different terms (name, ID, asset)

### Problem: Quick Process failed
**Solution:**
1. Check if request is in "Pending" status
2. Use individual actions instead:
   - Approve Only
   - Then Process Payment
   - Then Generate Tracking
3. If still failing, contact IT support

### Problem: Dropdown won't open
**Solution:**
1. Refresh the page (F5)
2. Clear browser cache
3. Try a different browser

### Problem: Receipt didn't download
**Solution:**
1. Check your browser's download folder
2. Allow popups for this site
3. Click "Download Receipt" again from dropdown
